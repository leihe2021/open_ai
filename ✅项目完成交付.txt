   e:\app\约血online

2️⃣ 双击运行
   一键部署.bat

3️⃣ 选择菜单
   [1] 自动安装依赖并打包

4️⃣ 等待完成
   约2-5分钟（取决于网络和硬件）

   dist\预约血.exe

📋 部署后使用
─────────────────────────────────────────────────────────────────

1. 将 dist 目录下的所有文件复制到目标电脑
2. 确保目标电脑为 Windows 10/11
3. 双击 "预约血.exe" 即可运行
4. 首次运行会自动创建数据库文件 records.db
5. 可开始使用血制品预约登记功能

⚠️ 注意事项
─────────────────────────────────────────────────────────────────

1. 系统要求
   • Windows 10 或 11
   • 建议内存: 4GB 以上
   • 磁盘空间: 至少 500MB

2. Python环境
   • 推荐版本: Python 3.9
   • 支持范围: Python 3.8-3.11
   • 不支持: Python 3.12+

3. 依赖安装
   • 首次打包需要网络连接
   • 建议关闭杀毒软件（避免误报）
   • 如安装失败，请查看错误提示

4. 数据安全
   • 数据库文件位于程序同目录
   • 建议定期备份 records.db
   • 可考虑数据加密存储
